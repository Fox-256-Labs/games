My D3.js Cookbook and Code Reference

This website serves as a reference / cook book for myself, when it comes to D3.js. But, it will also teach anyone the fundamentals of data visualization using D3.js.

Line Chart

How It Works:

1. Create an array of data points.

2. Use D3’s scale functions to normalize the data range.

3. Define a line generator function.

4. Append a path element to the SVG and set the 'd' attribute using the line generator.

Scatter Plot

How It Works:

1. Define an array of objects containing x and y coordinates.

2. Select an SVG and bind data to circle elements.

3. Set the cx and cy attributes dynamically based on data.

Bar Chart

How It Works:

1. Define a dataset as an array of values.

2. Select an SVG element and define the bar width and spacing.

3. Bind the data to rectangles using D3’s enter() method.

4. Set the position and size dynamically based on data values.
